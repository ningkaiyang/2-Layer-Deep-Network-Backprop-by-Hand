# Assignment 1: Deep Learning & Neural Networks (ICS 172B)

This repository contains files for **Assignment 1** of the course *Deep Learning & Neural Networks* (ICS 172B) at the
University of California, Irvine.

## Contents

- `1_Perceptron_and_Neural_Networks_Student.ipynb` – Jupyter notebook with solutions and explorations related to perceptrons
  and basic neural network architectures.
- `1_Perceptron_and_Neural_Networks_Student.html` – HTML export of the notebook for easy viewing.

## Description

The assignment explores the fundamentals of perceptrons and neural networks. It includes:

1. Implementing a perceptron learning algorithm.
2. Visualizing decision boundaries.
3. Building (implementing backpropogation by hand) and training simple 2-layer deep neural networks using first principles.
4. Analyzing network behavior with different activation functions and hyperparameters.
